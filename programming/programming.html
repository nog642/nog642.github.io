Languages

HTML

I know HTML isn't technically a programming language, but it is coding, so I'm going to put it here anyways. I learned some HTML when I was very young (around 5-7 years old) from my dad, who helped me make a website for myself. It was horribly designed, with a bright #0000FF background and ugly Times New Roman font, but at least I got the concept of tags and attributes.

Later, in 6th grade, I took a semester-long class called Digital Technology 1, where one of our assignments was to make a website. This one was a bit less terrible than the other one, and it was not very complete, but I got a refresher in HTML. Unlike the last website, where my dad did most of the coding, this time I did all of the work, and was able to do HTML myself. It was a bit better than the last one, with the background being light grey instead of bright blue. The font was still Times New Roman though, and the layout was actually worse.

The next year, when I was in 7th grade, I took Digital Technology 2, in which we were told to make a website again. This time, after using inspect element on the Google homepage, I found out about divs. I did some research, and learned about that and CSS. The website I made that time was much better than the previous two websites, I had a grey background with a very light grey box in the center, which contained all the text and a top menu. The box had a 1 px border, and the corners were rounded. Overall, it still looked pretty bad, but it was getting better. I tried to do a tutorial on jQuery, and one of my classmates made his menu in jQuery, but I didn't finish the tutorial.

In 8th grade, I took another semester-long class called Media Technology. Again, we had the assignment of making a website. I made the theme a bit darker than the last one, and got rid of the ugly box. I used CSS to make my top menu dropdown with many options. It was a big improvement, but similar to the other two years at school, most of the pages were empty because we had nothing to put there. This time, we put many of the projects we worked on there, but we didn't get all the way through what the teacher had planned at the beginning, so many of the pages were empty.

Near the end of 8th grade, my friend Ian Huang made a style guide called Orbit, and I decided to use it to make a website. Most of it was pretty easy, though I had to fiddle with the jQuery a lot to make the sidebar since I had never used jQuery before, and the final product was the website you're on now. Although what I really made was a website template, which Ian and I also reused for our biology project.

MATLAB

My dad taught me some MATLAB when I was very young (5-7 years old, like HTML), and it was the first real programming language I learned. It introduced me to for loops, while loops, if statements, how to use variables properly, keeping indexes, arrays, and all that. Most of the programs I wrote, I was helped by my dad, and he did most of the work. By the end though, I understood it quite well and could probably write a relatively simple program from start to finish. I haven't touched MATLAB in years though, and I've forgotten the syntax since then.

JavaScript

In 2015, when I was in 7th grade, I was taking Digital Technology 2. One of our assignments was to go on codecademy and take a tutorial in either JavaScript, Python, or Ruby. I chose to do JavaScript. Before this, the only real programming experience I had had was with MATLAB. Since I already had some programming background, learning JavaScript was relatively easy, and it was great after I had learned it. It was much more versatile than MATLAB, and I could make things that could run in a web browser. This was the first time I was able to make proper programs from beginning to end by myself.

Python

At first, since I chose to do a JavaScript tutorial instead of a Python one in 7th grade, I held that JavaScript was better, and I didn't want to learn Python. However, when I did my internship at Lightsphere AI, I had to learn it. Since the only other programming languages I had used were MATLAB and JavaScript, I didn't know what to expect. When I learned JavaScript, it was quite different from MATLAB. When I learned Python however, I was surprised by how similar it was to JavaScript—they could pretty much be translated one-to-one, line-by-line. In fact that's exactly what I did. To get a bit of practice with Python before I started any real projects, I converted my viginère cipher program, which I had written in JavaScript, to Python. After using Python for the majority of my work at Lightsphere AI, it's currently my strongest language.

Java

Java is the language I learned most recently. In mid-2016, I decided it was time for me to learn a new programming language. Java seemed the most reasonable option, since it was used very commonly. I started taking the Java tutorial on codecademy, and was pretty surprised. I thought it would be similar to JavaScript, or at least more similar than Python, but not at all. Python and JavaScript are much closer to eachother than to Java. I found out about static typing, which still seems like quite a limitation. I didn't finish the tutorial though, and forgot about it for a while.

During winter break of my freshman year of high school, I had an opportunity to earn some money doing work for Lightsphere AI. To fit in with the rest of the backend code though, I had to write my program in Java. I tried to finish the Java tutorial on codecademy, but quickly found that it wasn't proving very useful. Instead, I learned Java the natural way—just going at it, and looking stuff up on Google whenever necessary, usually clicking a link to stack overflow. I had already done some work with classes in Python, but only after having written a program in Java did I truly understand its object oriented nature.
